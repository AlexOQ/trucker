The Recommendation Algorithm

The advisor uses a greedy weighted optimization approach to recommend trailers. Here's how it works in plain terms:

  1. Build a cargo pool: We gather all cargo jobs available at depots in your chosen city. Each cargo has a value based on how much it pays per kilometer.
  2. Calculate trailer value: For each trailer type, we add up the total value of all cargo it can haul. A trailer that handles high-paying cargo gets a higher score.
  3. Pick the best trailer: The trailer with the highest total value gets selected first.
  4. Reduce remaining value: Once a trailer is selected, the cargo it covers becomes less valuable (since you already have a way to haul it). This prevents recommending 10 identical trailers.
  5. Repeat: Steps 2-4 repeat until your garage is full (default: 10 trailers).

Understanding the Score

The Score shown for each city and trailer represents potential earning power:

  • 0.8+ Excellent: Top-tier location or trailer. Maximum cargo value coverage.
  • 0.5 - 0.8 Good: Solid choice with strong cargo availability.
  • 0.3 - 0.5 Moderate: Decent option, but other choices may be better.
  • <0.3 Limited: Few high-value opportunities in this location.

Why Coverage % Can Be Misleading

You might notice a trailer with 44% coverage ranked below one with 19% coverage. This happens because:

  • Value matters more than variety: A trailer that hauls one very expensive cargo may score higher than one that handles many cheap cargoes.
  • Diminishing returns: If you already have trailers covering common cargo types, the algorithm values new coverage over redundant coverage.
  • Depot multipliers: If a city has multiple depots offering the same cargo, that cargo counts multiple times in the pool - making it more valuable to cover.
